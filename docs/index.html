Model Calibration Status auto-updated · 2026-06-03 16:32:09 UTC
▶ How calibration works
The calibration engine reads outcomes from every settled pick the model has produced (both displayed and shadow-only) and computes a per-sport, per-market realised vs predicted ratio. As sample size grows, each sport × market auto-promotes through phases without any manual intervention:
  • Phase 0 — < 100 settled picks: a credibility-weighted share of the target is applied (cred = settled ÷ 100), so a clear miss starts being corrected immediately and ramps up as data grows.
  • Phase A — full single ratio applied to every pick's edge (effective_edge = raw_edge × ratio); applied now equals target.
  • Phase B — per-bucket ratios (50–60% / 60–70% / 70–80% / 80%+) — ≥ 400 settled picks with ≥ 30 per bucket.
  • Phase C — isotonic regression smooth curve (reserved, future build-out).
Target vs Applied: Target is the full measured correction (realised ÷ predicted); Applied is what multiplies edges right now. They're equal from Phase A on; in Phase 0 Applied is a partial share of Target that grows toward it as more picks settle. Both shift daily as new results land.
The same effective edge is used to rank picks for the top-5 slots, so a noisier model whose 30% edges historically realise as 10% will have those edges shrunk automatically — no longer crowding out tighter-edge picks from sports with better-calibrated models.

Credibility Cap Auto-Adjustment auto-updated · 2026-06-03 16:31:12 UTC
▶ How auto-adjustment works
The system records every pick where the credibility cap fired (capping the model probability toward market). When a cap fires, the shadow log captures both the raw (uncapped) and capped probabilities — so once enough firings have settled, we can compare which prediction was closer to the actual outcome.
  • Value tuning: if raw was more accurate → widen +2.5%; if capped was more accurate → tighten -2.5%; otherwise keep.
  • Mode upgrade: after > 200 cap firings, the system also evaluates whether a smoother mathematical form (Mode 1 — tanh saturation) would have been more accurate than the current Mode 0 (hard clip). If tanh wins by > 1.5% MAE, the cap auto-promotes to Mode 1.
Safety bounds: never tighter than ±5.0%, never wider than ±30.0%. Value adjustments throttled to once per 30 days per cap; mode upgrades throttled to once per 60 days per cap. Minimum 50 cap firings required for value tuning, 200+ for mode upgrades.
